## Зависимости

* Kafka

## Сборка

Сборка производится с помощью Gradle

```shell
gradle build
```

## Запуск программы

### Поднять инфрастуктуру с зависимостями

```shell
docker-compose up -d
```

### Запустить приложение

```shell
java -jar .\build\libs\skillbox-order-service-0.0.1-SNAPSHOT.jar
```
